Primeros pasos

Primero se definen los estilos al nivel de etiqueta ul:
nav ul {} Se utilizan atributos como list-style, text-align, float y padding entre otros.

Continuamos a nivel de etiqueta elementos “li”, es decir, cada elemento de la lista. Mediante la propiedad “display” en modo bloque establecemos el tipo de caja para estos elementos, y con la propiedad “float” a la izquierda los distintos elementos se irán agrupando uno al lado de otro para formar el esquema final.

Para finalizar, a cada vínculo de la lista (es decir, las etiquetas tipo “a”) les eliminamos la decoración y le asignamos una tipografía. Además, mediante la propiedad “text-transform”, indicamos que el texto se transforme en mayúsculas (“text-transform” admite los valores: “uppercase”, “lowercase”, “capitalize” o “none”).

Por último, mediante la pseudo-clase “hover” creamos un estilo que actuará cuando situemos el ratón sobre el elemento. Las pseudo-clases :hover, :active, :link y :visited permiten al diseñador modificar el estilo de un elemento cuando el usuario realice una acción sobre el mismo. A continuación, se indican las acciones del usuario que activan cada pseudo-clase:

  • :link. se aplica a todos los vínculos que todavía no han sido visitados por el usuario.
  • :visited, se aplica a todos los vínculos que han sido visitados al menos una vez por el usuario.
  • :hover, el estilo creado se activa cuando el usuario sitúa el ratón encima de un elemento.
  • :active, el estilo creado se activa cuando el usuario pulsa con el ratón sobre un elemento. Se aplica durante un espacio de tiempo prácticamente imperceptible, ya que sólo dura desde que el usuario pulsa el botón del ratón hasta que lo suelta.
