Inicio

Nombre Uso Valores
color Cambia el color de las letras #000000 || black
font-size Cambia el tamaño de la fuente(las letras) 12pt || 1.3em
font-weight Cambia el peso de la fuente normal || bold
font-style Cambia el estilo de la fuente normal || italic
font-family Cambia la fuente Arial, Helvetica, sans-serif
margin Cambia el margen (puede usarse con top/bottom/left/right) 0 || 5px
padding Cambia el relleno/espaciado de los elementos 0 || 5px
border Añade o elimina los bordes de un elemento left, etc || 1px solid/shaded/dotted #color
background-color Cambia el color de fondo #color
text-align Alinea el texto center/left/right/justify
text-decoration Decora el estilo del texto underline/line-through
letter-spacing Cambia el espacio entre las letras 3px
line-height Cambia el espacio entre lineas 2em
width Cambia el ancho de un elemento 10px
float Define la flotación de un elemento, lo empuja a la derecha o izquierda left/right
clear Define el comportamiento de los laterales de un elemento both/left/right
display Define la forma de tratar un elemento(bloque,linea) block/inline
list-style Define un estilo de lista en una lista de elementos disc/outside/none/square
text-transform Permite la transformacion del texto en mayúsculas y minúsculas uppercase/lowercase/capitalize
