Etiquetas input

Tipos:

Dentro de el atributo "type" podemos poner:

  • text: para un campo de texto
  • password: para una contraseña
  • hidden: para introducir una caja de texto oculta
  • radio: para seleccionar un valor de un conjunto de elementos
  • checkbox: para seleccionar varios de un conjunto de elementos
  • button: muestra un boton (se usa con el atributo "onClick" para javaScript)
  • submit: muestra un boton, recarga la pagina y envia los datos del formulario por php
  • reset: muestra un boton, recarga la página y limpia los valores del formulario
